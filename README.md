# SPINE NPC/DDAS Tutorial Materials

Compact tutorial materials for a 2 to 2.5 hour hands-on SPINE session after a general introduction.

Start with [tutorials/README.md](tutorials/README.md).

The live path is:

1. Read reconstructed SPINE HDF5 output.
2. Learn truth matching and validation diagnostics.
3. Build and debug a detector-agnostic Michel-electron mini-analysis.
4. Learn the `spine-prod` production/configuration workflow.

The materials are designed for `ghcr.io/deeplearnphysics/spine:latest`.
